# Newsletter Sign Up Application 

A sample web application where the audience can sign up for the newsletter of the company. Once they sign up, the data such as email address and their name are stored at the MailChimp server.

## Built With

* framework used - Bootstrap.
* Styling - CSS.
* MailChimp - API used for storing the data.
* node/Express to create the server.
